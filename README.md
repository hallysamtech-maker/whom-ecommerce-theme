# Whom – Modern Multipurpose E-commerce HTML Template

Whom is a premium, modern, and fully responsive e-commerce HTML template designed for fashion stores, electronics shops, lifestyle brands, furniture stores, beauty products, and multipurpose online businesses.

Built with clean HTML5, CSS3, and JavaScript, Whom delivers a smooth shopping experience with elegant layouts, fast performance, responsive design, and conversion-focused UI components.

## Features

- Modern & Clean Design
- Fully Responsive Layout
- Dark & Light Mode Support
- Multipurpose E-commerce Template
- Fashion & Multi-product Ready
- Sticky Navigation
- Product Grid & Product Details Pages
- Shopping Cart UI
- Checkout Page
- Mobile Optimized
- Smooth Animations
- SEO Friendly Structure
- Fast Loading Performance
- Clean & Organized Code
- Easy Customization
- Cross-browser Compatible
